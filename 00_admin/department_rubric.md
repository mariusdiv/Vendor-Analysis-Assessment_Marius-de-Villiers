# Department Rubric

## Allowed Departments (12)
Engineering, Facilities, G&A, Legal, M&A, Marketing, SaaS,
Professional Services, Support, Finance, Product, Sales

## Classification Rules
- Assign by **function** (what the vendor does), not delivery model
- If a vendor spans multiple departments, use the **primary** function
- HR/People operations -> G&A or Professional Services
- IT/Security tools -> Engineering or SaaS (based on whether infrastructure or application)
- Insurance/benefits -> Support
- Travel/expenses -> G&A or Finance (based on whether tool or spend)

## Tie-Break Rules
1. Function over delivery model (e.g., legal consulting = Legal, not Professional Services)
2. Primary revenue driver over secondary usage
3. When equal, use the department with higher total spend concentration
